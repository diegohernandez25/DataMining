Install Node
------------
$ sudo apt-get update
$ curl -sL https://deb.nodesource.com/setup_10.x | sudo -E bash -
$ sudo apt install nodejs

Install Node Project Dependencies
---------------------------------
source install.sh

Run Project
-----------
node server.js

Browse Following web pages:
---------------------------
http://localhost:8081/index
http://localhost:8081/pois
http://localhost:8081/pois_map
http://localhost:8081/poi_loctype
